# QAMP 2025 Project 17: Exploring Quantum Key Distribution Protocols

https://github.com/qiskit-advocate/qamp-2025/issues/17

## Set-up
- Install uv package manager for python
- cd into the working directiory QAMP-2025-QKD
- Run: uv init (This command will create the pyproject.toml etc)
- Install sample packages like so: uv add qiskit

Refer: https://docs.astral.sh/uv/guides/projects/#managing-dependencies
